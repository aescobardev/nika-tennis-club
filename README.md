# Club de Tenis NIKA

Modern multi-page tennis club website built with HTML, Sass and JavaScript.

## Preview

A responsive sports club website focused on modern UI, accessibility, animations and interactive frontend components.

---

## Features

- Responsive multi-page layout
- Modern Sass architecture
- Mobile hamburger menu
- Dark mode with localStorage
- Scroll reveal animations
- FAQ accordion
- Membership modal
- Testimonial slider/carousel
- Advanced form validation
- localStorage persistence
- Semantic HTML and accessibility improvements
- SEO optimization

---

## Technologies

- HTML5
- Sass (SCSS)
- JavaScript (Vanilla JS)
- Responsive Design
- CSS Grid & Flexbox
